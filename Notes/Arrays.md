## 🧠 1. Array Traversal

---

### 📌 Definition
Array traversal means accessing each element of an array one by one using its index.

👉 In simple terms:
We start from the first element and move till the last element.

---

### 📦 Example Array

Index:   0   1   2   3  
Array:  [10, 20, 30, 40]

---

### 🔄 Traversal Flow

Step 1 → arr[0] = 10  
Step 2 → arr[1] = 20  
Step 3 → arr[2] = 30  
Step 4 → arr[3] = 40  

👉 We visit all elements sequentially

---

### 💻 Code

```java
for(int i = 0; i < arr.length; i++) {
    System.out.println(arr[i]);
