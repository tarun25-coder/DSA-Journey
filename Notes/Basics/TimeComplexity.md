# ⏱️ Time Complexity & Algorithms (Beginner Guide)

## 🧠 What is an Algorithm?

An algorithm is a step-by-step procedure to solve a problem.

### Example:
Find a number in an array:
1. Start from first element
2. Compare with target
3. If equal → return index
4. Else → move next

---

## ⚡ What is Time Complexity?

Time complexity tells how fast an algorithm runs as input size increases.

---

## 📊 Types of Time Complexity

### 🟢 O(1) — Constant Time
- Always takes same time

### 🟡 O(n) — Linear Time
- Time increases with input

### 🔵 O(log n) — Logarithmic Time
- Problem size reduces each step

---

## 🎯 Key Takeaways

- Algorithm = steps to solve problem
- Time complexity = speed of solution
